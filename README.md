### Docker Service Command
* Cek status :
`service docker status`
* Start jika terhenti :
`service docker start`
* Kalau uda selesai :
`service docker stop`
---
### Minikube Service Command
* Cek status :
`minikube status`
* Start jika terhenti :
`minikube start`
* Kalau uda selesai :
`minikube stop`
